# s3goadmin
An S3 Admin utility witten in Goland

The utility has following features:

- Create Bucket
- List Buckets
- Delete Bucket
- Upload Object
- Download Object
- Delete Object
- List Objects
- Search Objects using prefix & delimited
- Folder navigation

Need to install the aws dependencies to start

 `go get -u github.com/aws/aws-sdk-go`
